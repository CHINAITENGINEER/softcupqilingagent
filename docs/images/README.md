# SafeOps Console screenshots

Place curated, non-sensitive product screenshots in this directory after visual QA. The documentation and portfolio pages should reference these filenames:

- `safeops-console-overview.png`
- `safeops-console-chat.png`
- `safeops-console-approval.png`
- `safeops-console-audit.png`
- `safeops-console-rag.png`

Do not capture API keys, user tokens, internal hostnames, or real production audit records in screenshots.
